#### Importing packages and reading image:
```Python
import cv2
import matplotlib.pyplot as plt
import numpy as np
img=cv2.imread("blue.jpg")
img=cv2.cvtColor(img,cv2.COLOR_BGR2RGB)

def plot(img1,nam1,img2,nam2):
  plt.figure(figsize=(9,9))
  plt.subplot(1,2,1)
  plt.imshow(img1)
  plt.title(nam1)
  plt.axis("off")
  plt.subplot(1,2,2)
  plt.imshow(img2)
  plt.title(nam2)
  plt.axis("off")
  plt.show()
```
